# Toy Pipeline Output Examples

This directory contains **pre-generated toy examples** showing what OpenAMP Foundry pipeline outputs look like.

**IMPORTANT**: All candidate IDs use the `TOY-` prefix. All sequences are made up and do NOT represent real antimicrobial peptide candidates. These files exist solely to show output format and structure.

## Files

| File | Description |
|------|-------------|
| `toy_ranked_example.jsonl` | 3 toy ranked candidates (one JSON object per line) |
| `toy_certificate_example.json` | Example evidence certificate for a toy candidate |

## What These Show

- Field names and value ranges in ranked output
- Evidence certificate structure and required fields
- The claim boundary notice format
- How dry-lab-only status is encoded

## What These Are NOT

These are NOT real candidates. They are NOT nominations for synthesis. They exist only to illustrate output format.
