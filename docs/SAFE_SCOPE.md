# Safe Scope

This project is limited to computational candidate triage and evidence generation.

## Allowed

- sequence validation;
- feature extraction;
- transparent scoring;
- novelty checking;
- synthesis feasibility screening;
- evidence certificate generation;
- benchmark design;
- responsible release policy;
- discussion of independent assay as a future validation step at a high level.

## Not allowed in this repository

- culturing protocols;
- pathogen handling instructions;
- organism modification procedures;
- instructions for improving harmful biological traits;
- toxicity-maximization objectives;
- dangerous organism assay protocols;
- release of unscreened high-risk candidate lists;
- model weights designed to optimize harmful biological effects.
