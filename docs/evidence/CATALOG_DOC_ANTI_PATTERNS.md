# Documentation Anti-Patterns Catalog

Common anti-patterns to avoid.

| Anti-Pattern | Problem | Solution |
|-------------|---------|----------|
| Vanishing source | A doc describes a feature that no longer exists | Keep docs in sync with code |
| Copy-paste drift | Same info in multiple places, gets out of sync | Single source of truth, cross-reference |
| Zombie doc | Deprecated doc still linked from navigation | Archive and update links |
| Claim without caveat | States a result without its limitations | Always pair claims with caveats |
| Doc as dumpster | One doc accumulates everything | Split into focused documents |
| No owner | No one is responsible for keeping it current | Assign an owner |
