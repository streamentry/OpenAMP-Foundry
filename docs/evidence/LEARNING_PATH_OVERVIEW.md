# Learning Path Overview

The recommended order to learn OpenAMP Foundry.

```
1. Read AGENTS.md and CLAUDE.md (30 min)
2. Run `make demo` and explore outputs (30 min)
3. Read LIMITATIONS_OVERVIEW.md (15 min)
4. Read QUICKSTART.md and run through it (15 min)
5. Read CONTRIBUTOR_CURRICULUM.md and pick a task (15 min)
6. Read ARCHITECTURE.md (30 min)
7. Read EVIDENCE_CERTIFICATE.md (15 min)
8. Read BENCHMARK_GOVERNANCE.md (15 min)
9. Read PROOF_LADDER.md (10 min)
10. Choose a learning path:
    - Reviewer: CURRICULUM_REVIEWER.md
    - Benchmark reviewer: CURRICULUM_BENCHMARK_REVIEWER.md
    - Release operator: CURRICULUM_RELEASE_OPERATOR.md
    - Agent: CURRICULUM_AGENT.md
```
