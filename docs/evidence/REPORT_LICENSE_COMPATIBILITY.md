# License Compatibility Report

License compatibility for project dependencies.

| Dependency | License | Compatible with Apache 2.0? |
|------------|---------|:---------------------------:|
| pytest | MIT | ✅ |
| ruff | MIT | ✅ |
| PyYAML | MIT | ✅ |
| jsonschema | MIT | ✅ |

## Rules
- Apache 2.0 is compatible with MIT, BSD, and Apache 2.0.
- GPL and AGPL are NOT compatible and must not be used.
- LGPL may be acceptable with review.
- All dependencies must have a documented license.
- If a dependency's license is unknown, do not use it.
