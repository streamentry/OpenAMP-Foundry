# Machine-Readable Report Manifest

The report manifest is generated by:

```bash
make full-reproducibility-report
```

The JSON output at `outputs/full_reproducibility_report.json` includes:

## Top-Level Structure
```json
{
  "report_type": "full_reproducibility",
  "pipeline_version": "0.1.0",
  "generated_at": "ISO 8601 timestamp",
  "loops_completed": 50,
  "phases_completed": { "...": true },
  "sections": {
    "git": { "head_sha": "...", "recent_commits": [...] },
    "tests": { "collection_output": "..." },
    "benchmarks": { "...": { "auroc": 0.78 } },
    "simulation_modules": { "...": "experimental" },
    "phase_4_readiness": { "artifacts": { "...": "✅" } },
    "limitations": { "...": "No wet-lab data exists" }
  }
}
```

## Consuming the Report
The JSON report can be parsed by any JSON-capable tool for automation or dashboards.
