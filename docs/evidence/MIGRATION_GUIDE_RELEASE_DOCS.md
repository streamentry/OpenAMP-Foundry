# Migration Guide: Release Docs

When preparing a release, ensure these docs are updated:

1. `docs/evidence/METRICS_CURRENT.md` — update benchmark numbers.
2. `docs/research/ROADMAP.md` — update completed items.
3. `docs/evidence/SIMULATION_BENCHMARK.md` — update gate verdict.
4. `docs/review/PUBLICATION_PACK.md` — complete checklist.
5. `README.md` — update version references.
