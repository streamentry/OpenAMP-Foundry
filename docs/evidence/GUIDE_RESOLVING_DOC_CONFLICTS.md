# Guide: Resolving Doc Conflicts

When two documents give contradictory guidance.

## Steps
1. Identify the authoritative source for the topic.
2. If neither is authoritative, use the more recently updated document.
3. Update the less authoritative doc to match.
4. Add cross-references in both docs noting the resolution.
5. If the conflict affects policy, log it in the decision log.
