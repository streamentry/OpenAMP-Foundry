# Proof Ladder — Concept Card

The proof ladder defines evidence levels for claims made about pipeline outputs.

## Levels

| Level | Name | Meaning |
|-------|------|---------|
| 0 | Syntax valid | Sequence uses canonical amino acids |
| 1 | Reproducible features | Features are deterministic |
| 2 | Baseline score | Heuristic passes basic filters |
| 3 | Multi-model consensus | Independent predictors agree |
| 4 | Leakage-resistant benchmark | Recovers hidden positives |
| 5 | Lab assay | Independent biological test |
| 6 | Independent replication | Second lab reproduces result |

## Key Rule

Computational scores (levels 0–4) are not biological proof. Only levels 5–6
constitute biological evidence.
