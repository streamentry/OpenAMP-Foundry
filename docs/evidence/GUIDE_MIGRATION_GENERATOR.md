# Artifact Migration Guide Generator

Template for creating artifact migration guides.

## Template
```markdown
# Migration Guide: [Old Format] → [New Format]

## What Changed
Brief description of the format change.

## Why
Why the change was necessary.

## Migration Steps
1. Step-by-step instructions.
2. Commands to run.
3. Verification steps.

## Backward Compatibility
- Old format supported until: [date or version]
- Converter available: [path or command]

## Related
- [Link to schema changelog]
- [Link to deprecation policy]
```

## Current Status
No migrations are currently needed. This template is for future use.
