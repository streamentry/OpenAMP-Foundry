# Documentation Maintenance Calendar

Regular maintenance tasks for documentation.

## Weekly
- [ ] Review and merge documentation PRs
- [ ] Check for broken links (run `check_doc_links.py`)

## Monthly
- [ ] Review new issues tagged `docs` and assign priorities
- [ ] Update METRICS_CURRENT.md if benchmarks changed
- [ ] Check QUICKSTART.md commands still work

## Quarterly
- [ ] Full documentation audit (run all audit checklists)
- [ ] Review stale documents and archive or update
- [ ] Update learning paths and curricula
- [ ] Run `check_claims.py` on all public-facing docs

## Annually
- [ ] Review documentation governance charter
- [ ] Review safety documentation
- [ ] Archive documents that are no longer relevant
