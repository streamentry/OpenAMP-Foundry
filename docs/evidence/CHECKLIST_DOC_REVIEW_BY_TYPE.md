# Doc Review Checklist by Change Type

## New Document
- [ ] Has a clear purpose statement
- [ ] Is linked from PROJECT_INDEX.md
- [ ] Passes `check_claims.py`
- [ ] Passes `check_doc_links.py`
- [ ] Has the correct metadata header

## Document Update
- [ ] Changes are accurate
- [ ] Links still resolve
- [ ] No overclaiming language introduced
- [ ] Metadata updated (Last updated date)

## Document Deprecation
- [ ] Deprecation notice added at top
- [ ] Cross-references updated
- [ ] Archive path documented in archive index
