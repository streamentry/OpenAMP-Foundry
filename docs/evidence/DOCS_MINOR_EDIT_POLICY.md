# Documentation Minor-Edit Policy

Minor edits can be made without full review.

## What Counts as Minor

- Typo fixes
- Link corrections
- Formatting improvements (tables, lists, headings)
- Updating version numbers
- Clarifying wording without changing meaning

## What Requires Full Review

- Adding or removing sections
- Changing policy statements
- Adding or removing claims
- Restructuring documents

## Process

Minor edits can be committed directly to `main` or submitted as a PR with
the `minor` label. Review is optional but encouraged.
