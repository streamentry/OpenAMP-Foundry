# Tutorial Prerequisites Checklist

Before starting a tutorial, ensure:

- [ ] Python 3.11+ is installed
- [ ] `git` is installed
- [ ] A virtual environment is created: `python3 -m venv .venv`
- [ ] Dependencies are installed: `pip install -e ".[dev]"`
- [ ] The tutorial's estimated time is accurate
- [ ] All commands in the tutorial work from a clean checkout
