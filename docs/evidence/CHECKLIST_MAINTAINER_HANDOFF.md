# Maintainer Handoff Checklist

When a maintainer is unavailable or leaving, use this checklist.

- [ ] Identify all PRs the maintainer is reviewing.
- [ ] Reassign PRs to other maintainers.
- [ ] Identify all issues the maintainer is assigned to.
- [ ] Reassign or unassign issues.
- [ ] Document any in-progress work the maintainer was doing.
- [ ] Update the folder ownership map if needed.
- [ ] Notify the team of the change.
- [ ] Update the decision log.
