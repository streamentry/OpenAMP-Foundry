# Public FAQ for Non-Specialists

## What is OpenAMP Foundry?
A computational pipeline that ranks antimicrobial peptide candidates before lab testing.

## Does it discover new antibiotics?
It nominates candidates for testing. Discovery requires wet-lab validation.

## Are the scores reliable?
They are predictions, not proof. The pipeline is benchmarked against known AMPs but has limitations (charge bias, no wet-lab data).

## Can I use the results for my research?
Yes, with the understanding that all results are computational predictions.

## Is it safe?
The pipeline is dry-lab only — no biological experiments are conducted. All code is open source (Apache 2.0).
