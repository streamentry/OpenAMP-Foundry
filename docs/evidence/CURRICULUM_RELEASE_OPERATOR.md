# Release Operator Curriculum

For contributors who prepare and execute releases.

## Prerequisites
- Complete the Reviewer Curriculum

## Skills
- Can run `make full-reproducibility-report` and interpret the output
- Can run all pre-release checks (doc links, claims, tests)
- Can build the lab batch pack
- Can complete the PUBLICATION_PACK.md checklist
- Understands the release status definitions
- Can create a release tag and publish release notes
