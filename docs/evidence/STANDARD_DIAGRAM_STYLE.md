# Diagram Source Style Guide

- Use Mermaid for diagrams (flowchart, sequence, component)
- Keep diagrams in markdown code blocks with the mermaid language tag
- Place diagrams near the relevant text, not in a separate file
- Label all nodes clearly
- Limit to 15 nodes per diagram
- Use `flowchart TD` for top-down, `flowchart LR` for left-right
- Use `-->` for directed edges, `---` for undirected
