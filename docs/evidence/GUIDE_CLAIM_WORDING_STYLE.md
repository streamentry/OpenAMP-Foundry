# Claim Wording Style Guide

How to write about computational results without overclaiming.

## Do Use
- "Computationally nominated candidate"
- "Predicted antimicrobial peptide"
- "Dry-lab candidate"
- "Selected by reproducible pipeline"
- "Evidence package"
- "Benchmark-supported under stated assumptions"

## Don't Use
- "Drug candidate" — requires wet-lab evidence
- "Proven antimicrobial" — computation does not prove
- "Safe in humans" — requires clinical evidence
- "Validated by computation" — computation triages, doesn't validate
- "Breakthrough therapy" — requires clinical evidence

## Rule of Thumb
If you wouldn't say it about a random sequence without running any tests,
you shouldn't say it about a computationally nominated candidate.
