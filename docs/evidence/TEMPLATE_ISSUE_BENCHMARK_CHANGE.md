# Issue Template for Benchmark Changes

Template for issues that propose benchmark changes.

```markdown
## Summary
Brief description of the benchmark change.

## Current Benchmark
Link to current benchmark card and metrics.

## Proposed Change
What benchmark is being added, modified, or removed.

## Rationale
Why this change is needed.

## Metrics
What metrics will be affected.

## Gate Impact
Will this change affect the benchmark regression gate?

## Related
- Link to related PRs or discussions.
```

## Rules
- Benchmark changes affecting the gate require maintainer review.
- New benchmarks must include documented caveats.
- Benchmark changes should be discussed before implementation.
