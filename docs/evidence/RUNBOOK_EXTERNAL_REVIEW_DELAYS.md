# Maintainer Runbook: External Review Delays

When an external review is delayed:

1. Check the review request for completeness (all required fields present).
2. Follow up with the reviewer — allow 1 week between follow-ups.
3. If the reviewer is unavailable, identify an alternative reviewer.
4. If no alternative is available, escalate to the project lead.
5. Document the delay and resolution in the decision log.
