# Reviewer Conflict-of-Interest Note

Reviewers should declare any conflict of interest before reviewing.

## Examples of Conflicts
- You authored the PR being reviewed
- You have a personal relationship with the author
- You have a financial interest in the outcome
- You are competing with the project in the same space

## Process
1. Declare the conflict in the PR comments.
2. If the conflict is significant, recuse yourself and request a different reviewer.
3. If the conflict is minor, document it and proceed with review.
