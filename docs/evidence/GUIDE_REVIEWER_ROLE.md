# Reviewer Role Guide

Responsibilities of different reviewer roles.

| Role | Can Approve | Required For |
|------|:-----------:|--------------|
| Contributor | No | Can comment, suggest changes |
| Reviewer | Low-risk PRs | Medium-risk PRs |
| Maintainer | High-risk PRs | Policy, safety, calibration changes |
| Safety reviewer | Safety changes | Safety policy updates |
| Project lead | All changes | Final approval on releases |

## Guidelines
- Reviewers should match their expertise to the change.
- Safety changes always require a safety reviewer.
- High-risk changes require at least 2 reviewers.
- The project lead has final authority on disputed reviews.
