# Beginner Glossary Path

Recommended order to learn project terminology.

1. Candidate — a peptide sequence being evaluated
2. Score — a numerical prediction of a property
3. Ensemble — weighted combination of individual scores
4. Benchmark — a test of pipeline performance against known data
5. Gate — a safety check that blocks or allows an action
6. Calibration — adjusting scores based on real data
7. Evidence certificate — machine-readable record of a candidate's scores
8. Manifest — record of inputs and outputs for a pipeline run
9. Ablation — removing a component to measure its contribution
10. Concept card — short explanation of a single concept
