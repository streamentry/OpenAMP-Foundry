# Agent Contributor Curriculum

For AI agents contributing to the repository.

## Prerequisites
- Read AGENTS.md (primary operating contract)
- Read AGENT_TASKS.json (safe task routing)

## Stage 1: Safe tasks (docs, tests, schemas)
- Fix broken doc links
- Add tests for existing functionality
- Improve documentation clarity
- Add schemas for artifacts

## Stage 2: Normal tasks (CI, benchmarks)
- Improve CI workflows
- Add benchmark scripts
- Add concept cards
- Add runbooks and checklists

## Stage 3: Review-required tasks
- Changes to calibration pipeline
- Changes to safety policy
- Changes to scoring weights
- Any change that affects claims or releases

## Forbidden
- Releasing candidates without human review
- Changing safety policy without human review
- Making scientific claims without evidence
