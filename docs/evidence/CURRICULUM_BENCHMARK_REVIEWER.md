# Benchmark Reviewer Curriculum

For reviewing benchmark-related changes.

## Prerequisites
- Complete the Reviewer Curriculum first

## Skills
- Can interpret AUROC, AUPRC, recall, and precision metrics
- Understands charge-inflation caveat
- Can assess whether a benchmark regression is significant
- Can verify that benchmark gates pass
- Can assess whether a new benchmark has documented caveats
