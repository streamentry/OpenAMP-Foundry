# Environment Assumption Standard

Every document should state its assumptions about the reader's environment.

## Required
- Python version (minimum: 3.11)
- Operating system (tested on: macOS, Linux)
- Working `git` installation
- Working `make` installation

## Optional but recommended
- Virtual environment manager (venv)
- RAM (minimum: 4 GB, recommended: 8 GB)
- Disk space (minimum: 500 MB)
- Network access (required for `pip install`, optional for local checks)
