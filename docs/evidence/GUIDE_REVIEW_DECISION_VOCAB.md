# Review Decision Vocabulary

Standard terms for review decisions.

| Decision | Meaning | Action Required |
|----------|---------|----------------|
| Approved | Ready to merge | Merge the PR |
| Approved with suggestions | Ready but has optional improvements | Consider suggestions |
| Changes requested | Must address before merge | Make changes, re-request review |
| Blocked | Cannot proceed | Resolve the blocking issue first |
| Deferred | Postponed to a later milestone | Close without merging |

## Rules
- "Approved" with no changes means merge immediately.
- "Changes requested" means re-review is required.
- "Blocked" requires escalation to maintainers.
