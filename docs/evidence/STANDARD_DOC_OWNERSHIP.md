# Documentation Ownership Metadata

Every document should have ownership metadata in its header.

## Format
```markdown
> **Owner:** @github_username or team name
> **Last reviewed:** YYYY-MM-DD
> **Status:** Draft | Internal | Public-safe
```

## Rules
- Every document has exactly one owner.
- The owner is responsible for keeping the document current.
- If the owner changes, update the metadata.
- Documents without an owner are owned by the project lead by default.
