# Archived Artifact Index

Index of archived artifacts and their locations.

## Index Format
| Artifact | Archive Path | Date Archived | Superseded By |
|----------|-------------|:-------------:|:-------------:|
| — | — | — | — |

## Rules
- When an artifact is archived, add it to this index.
- Include the date and reason for archiving.
- If superseded, link to the replacement artifact.
- Archived artifacts should be moved to `archive/` directories.
- The index should be checked before creating new artifacts to avoid duplication.
