# Release Status Vocabulary

Standard terms for release statuses.

| Status | Meaning | Can be cited? |
|--------|---------|:-------------:|
| Draft | Work in progress | No |
| Internal | Project team reviewed | With caveats |
| Review-only | Shared with reviewers | In review context |
| Public-safe | Safe for public sharing | Yes, with limitations |
| Restricted | Sensitive content | No |
| Archived | Superseded | As historical reference |
| Deprecated | No longer maintained | Do not cite |
