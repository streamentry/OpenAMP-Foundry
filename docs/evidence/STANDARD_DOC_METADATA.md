# Documentation Metadata Format

Each document should include metadata at the top.

## Format
```markdown
> **Purpose:** One-sentence description of the document's purpose.
> **Status:** Draft | Internal | Review-only | Public-safe | Archived
> **Last updated:** YYYY-MM-DD
> **Owner:** Role or GitHub username
```

## Example
```markdown
> **Purpose:** Defines how documentation is governed in the project.
> **Status:** Internal
> **Last updated:** 2026-07-08
> **Owner:** Maintainer
```
