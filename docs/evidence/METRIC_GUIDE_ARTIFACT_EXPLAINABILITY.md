# Metric Guide: Artifact Explainability

**Question:** Can a reviewer understand what a given artifact means and how it was produced?

**What to track:** Whether each artifact type has a corresponding concept card or documentation section.

**Target:** Every artifact type in ARTIFACT_MAP.md has a concept card.

**What to measure:** Count of artifact types with concept cards divided by total artifact types.

**What good looks like:** 100% coverage.
