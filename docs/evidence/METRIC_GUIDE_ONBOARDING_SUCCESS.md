# Metric Guide: Onboarding Success

**Question:** How quickly can a new contributor go from zero to running the demo?

**What to track:** Time from `git clone` to `make demo` completing successfully.

**Target:** < 15 minutes per QUICKSTART.md.

**How to measure:** Run through QUICKSTART.md with a stopwatch.

**What good looks like:** A contributor with Python installed can complete the full path in one sitting without asking for help.
