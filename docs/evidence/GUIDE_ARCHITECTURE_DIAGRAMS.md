# Architecture Diagram Source Files

Source files for pipeline architecture diagrams.

| Diagram | Source | Format |
|---------|--------|--------|
| Artifact flow | docs/evidence/DIAGRAM_ARTIFACT_FLOW.md | Mermaid |
| Data flow | docs/evidence/DIAGRAM_DATA_FLOW.md | Mermaid |
| Dependency graph | docs/evidence/DIAGRAM_DEPENDENCY_GRAPH.md | Mermaid |

## How to Update
1. Edit the Mermaid diagram in the source file.
2. Preview the diagram using a Mermaid renderer.
3. Verify the diagram renders correctly in GitHub (native support).

## Rules
- All diagrams should use Mermaid format.
- Keep diagrams focused — no more than 15 nodes.
- Label all nodes and connections clearly.
- Update related text when a diagram changes.
