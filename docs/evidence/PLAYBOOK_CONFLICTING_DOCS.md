# Scenario Playbook: Conflicting Docs

**Scenario:** Two documents give contradictory guidance.

## Steps

1. Identify which document is the authoritative source for the topic. Check `docs/evidence/METRICS_CURRENT.md` (metrics) or `docs/evidence/EVIDENCE_CERTIFICATE.md` (evidence).
2. If neither is authoritative, determine which document is more recently updated or more specific.
3. Update the less authoritative document to match the authoritative one.
4. Add a cross-reference in both documents noting the resolution.
5. If the conflict affects policy, log the resolution in the decision log.
