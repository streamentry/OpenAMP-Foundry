# Review Packet Overview

A review packet contains everything a reviewer needs to evaluate a claim.

## Required Contents
- Candidate list with scores
- Evidence certificates for selected candidates
- LIMITATIONS_OVERVIEW.md
- The specific claim or question being reviewed

## Optional Contents
- METRICS_CURRENT.md for benchmark context
- Simulation scores (if used)
- Expert review template

## Rules
- Packets must not overclaim.
- Packets must state which evidence is computational vs biological.
- Reviewers must be told this is a dry-lab nomination only.
