# Scenario Playbook: Proof Confusion

**Scenario:** Someone cites a computational score as biological proof.

## Steps

1. Refer them to `docs/evidence/PROOF_LADDER.md` — levels 0-4 are computational.
2. Explain that only levels 5 (lab assay) and 6 (independent replication) constitute biological evidence.
3. If the confusion is in project documentation, update the document to clarify the proof level.
4. If the confusion is in an external communication, provide a correction.
5. Log the incident in the decision log as a documentation clarification.
