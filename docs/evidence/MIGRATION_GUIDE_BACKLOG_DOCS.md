# Migration Guide: Backlog Docs

When moving items from backlog to active or completed:

1. Update `docs/research/NEXT_100_PR_MAP.md` if the item was there.
2. Update `docs/research/ROADMAP.md` with the new status.
3. If the work was completed, ensure all related docs are updated.
4. Close the corresponding GitHub issue with a link to the PR.
