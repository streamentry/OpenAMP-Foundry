# Citation and Source Policy

How to cite sources in documentation.

## Required Citations
- All dataset sources (with license information)
- All algorithms or methods not original to this project
- All benchmark references
- All literature claims not supported by project evidence

## Citation Format
Use inline links for web resources and footnotes for academic references.

## Source Requirements
- Sources must be publicly accessible where possible.
- Paywalled sources should be noted as such.
- Sources must be specific (page number, section, or timestamp).
- "Personal communication" is not an acceptable citation.
