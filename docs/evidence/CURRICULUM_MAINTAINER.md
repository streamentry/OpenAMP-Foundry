# Maintainer Curriculum

For project maintainers.

## Prerequisites
- Complete the Reviewer Curriculum

## Skills
- Can review PRs of any risk class
- Can merge PRs and manage the release process
- Can resolve documentation disputes (see ESCALATION_POLICY.md)
- Can approve new document creation
- Can manage issues and milestones
- Understands the full project architecture
- Can assess whether a change needs design review first
