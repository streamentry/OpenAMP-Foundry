# Contributor Runbook: Picking Up an Issue

1. Find an open issue labelled `good-first-issue` or `help-wanted`.
2. Comment on the issue that you're working on it.
3. Ask clarifying questions if the scope is unclear.
4. Create a branch from `main`.
5. Make your change, add tests, verify nothing is broken.
6. Create a PR that references the issue number.
