# Responsible Use

You may use this project to:

- evaluate peptide candidates computationally;
- build reproducible benchmarks;
- generate auditable evidence certificates;
- prepare candidates for qualified expert review;
- publish safe negative and positive scientific findings.

You may not use this project to:

- design harmful biological agents;
- optimize toxicity, delivery, persistence, immune evasion, or pathogenicity;
- bypass safety review;
- claim biological activity without experimental validation;
- market candidates as treatments;
- use project outputs as clinical advice.

This document is a governance policy, not a substitute for law, institutional review, biosafety review, or domain expert review.
